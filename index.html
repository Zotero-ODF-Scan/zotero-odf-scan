Zotero Tools

This is a small collection of supplementary tools for use with Zotero.

Scannable Cite translator
This is an export translator for Zotero, intended for use as a Default Export Format with the QuickCopy function. It generates a simple cite with parsable markup and a link to the Zotero item, which can be converted to a live Zotero citation using RTF/ODF Scan for Zotero plugin.
RTF/ODF Scan for Zotero
This plugin is a companion to the Scannable Cite translator, suitable for use with Zotero or Multilingual Zotero. It adds the ability to convert Scannable Cite markers in an ODF document (the native format of LibreOffice) to fully functional Zotero citations. Conversion works in both directions, and documents can contain a mixture of Scannable Cite markers and "live" Zotero citations. Conversion will transform all cites in the document to the selected form. When converting to live citations, adjacent markers will be merged into "multiple" Zotero citations.
